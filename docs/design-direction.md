# Design direction

## Kierunek wizualny

Nowoczesny industrial B2B.

## Cechy

- ciemne lub jasne neutralne tła,
- akcent inspirowany energią lasera: pomarańcz / bursztyn / czerwień,
- mocna typografia,
- dużo przestrzeni,
- wysoka czytelność,
- zero generycznych gradientów w stylu startup AI.

## Inspiracje

- przemysłowe strony producentów maszyn,
- premium B2B software layout,
- porównania produktów w stylu pricing table,
- czytelne sekcje edukacyjne.

## Zasady

- lewy align jako default,
- maksymalnie klarowna hierarchia treści,
- responsywność mobile-first,
- sekcje muszą prowadzić do kontaktu lub porównania modeli.
