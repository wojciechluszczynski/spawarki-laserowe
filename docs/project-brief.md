# Project brief

## Projekt

Spawarkilaserowe.com — nowoczesny serwis B2B prezentujący 3 modele spawarek laserowych, z funkcją porównania modeli, blogiem SEO oraz prostym panelem do zarządzania treścią.

## Kontekst biznesowy

Klient potrzebuje dwóch bliźniaczych serwisów:
- spawarkilaserowe.com
- spawarkilaserowe.com

Na start ten pakiet dotyczy wyłącznie spawarkilaserowe.com, ale architektura i treści mają być możliwie łatwe do skopiowania do drugiego serwisu.

## Główny cel strony

- wyjaśnić czym są spawarki laserowe,
- pokazać zalety rozwiązania,
- pomóc klientowi zrozumieć różnice między 3 modelami,
- generować leady,
- budować widoczność organiczną przez SEO i blog.

## Grupa docelowa

- właściciele firm produkcyjnych,
- dyrektorzy produkcji,
- technolodzy,
- osoby odpowiedzialne za inwestycje w park maszynowy,
- odbiorcy B2B, którzy chcą szybko porównać rozwiązania.

## Styl komunikacji

- konkretny,
- techniczny, ale zrozumiały,
- bez lania wody,
- premium, nowocześnie, przemysłowo,
- wiarygodny i spokojny.

## Funkcje MVP

- landing page,
- sekcje edukacyjne,
- porównanie 3 modeli,
- FAQ,
- formularz kontaktowy,
- blog,
- panel CMS do edycji wpisów i metadanych SEO.

## Technologie

- frontend: Next.js + React + TypeScript + Tailwind CSS,
- deployment: GitHub + hosting klienta,
- backend/CMS: pliki lokalne lub Supabase, jeśli potrzebne.
