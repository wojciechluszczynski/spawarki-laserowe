# Refactor rules

## Główne reguły

- nie mieszaj contentu z komponentami,
- nie powielaj sekcji dla drugiego serwisu — przygotuj warstwę konfiguracyjną,
- trzymaj metadata w osobnych helperach,
- unikaj nadmiarowych zależności,
- preferuj pliki konfiguracyjne json/md nad twardym kodowaniem treści.
